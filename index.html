1. Введение

Мы уже изучили основы синтаксиса JavaScript, узнали языковые конструкции switch и if…else. Научились работать с примитивными типами данных и взаимодействовать с пользователем.

В этом модуле вы познакомитесь с циклами, объектами и массивами. Это сложные структуры данных, с которыми вам предстоит постоянно работать в реальных проектах.

Начнём с циклов. Узнаете, как они помогают сократить написание однообразного кода.

Затем разберётесь, что такое массивы и научитесь с ними работать.

После перейдёте к объектам. Это важная структура данных для любого разработчика, потому что данные с сервера и обратно чаще всего отправляют в виде объектов.

И закончите модуль изучением таких структур данных, как Map и Set.

А пока повторите пройденный материал.

Задание 1.1

Будет ли выведен результат в консоль?

const first = true;

const second = 2;

if (first > 2) {

console.log('Yes');

}

Нет, не будет.

Ответ

Верно:

Значение не будет выведено в консоль. При таком условии переменная first будет приведена к типу number и мы получим значение 1. Условие в скобках выдаст false. Блок кода с вызовом консоли не будет выполнен.

Задание 1.2

Есть переменная x, которая может принимать любое значение. Напишите программу, которая в зависимости от типа данных x выводит в консоль сообщение вида: «x — число».

Опишите три случая, когда x равно числу, строке или логическому типу. В других случаях выводите сообщение: «Тип x не определён».

Использовать prompt в задаче не нужно.

Вариант 1: Входные данные: x = 5 Выходные данные: х — число
Вариант 2: Входные данные: х = “Строка” Выходные данные: х — строка
Вариант 3: Входные данные: x = true Выходные данные: х — логический тип
Вариант 4: Входные данные: х = null Выходные данные: Тип х не определён
Посмотреть ответ

// Решить можно несколькими способами

// Объявление переменной общее для всех способов

const x = 5;

// или

const = "Строка";

// или

const x = true;

// Решение 1

switch (typeof x) {

case "number":
console.log("x — число");
break;
case "string":
console.log("х — строка");
break;
case "boolean":
console.log("х — логический тип");
break;
default
console.log(" Тип х не определён");

}

// Решение 2

if (typeof x === "number") {

console.log("x — число");

} else if (typeof x === "string") {

console.log("х — строка");

} else if (typeof x === "boolean") {

console.log("х — логический тип");

} else {

console.log(" Тип х не определён");

}

Задание 1.3

Есть две переменные — х и y. Присвойте переменной х значение «Чётное число» или «Нечётное число» в зависимости от переменной y.

Решение должно быть написано в одну строку.

ответ:
x = y % 2 === 0 ? "Чётное число" : "Нечётное число";
